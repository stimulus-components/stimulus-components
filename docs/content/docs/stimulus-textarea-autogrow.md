## Usage

::code-block{tabName="app/views/index.html"}

```html
<textarea data-controller="textarea-autogrow">
Very long text here.
</textarea>

<textarea data-controller="textarea-autogrow" data-textarea-autogrow-resize-debounce-delay-value="500">
Very long text here.
</textarea>
```

::

## Configuration
